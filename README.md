# 亚运会志愿者管理前端项目(PC 版)

## 技术栈

- vue 3.2
- vuex 4.0
- vue-router 4.0
- element-plus 2.1
- axios
- webpack
- ES6
- Mock.js

## 项目运行

### Install dependencies

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```
